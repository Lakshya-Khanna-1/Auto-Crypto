# Auto-Crypto Algorithmic Trading Platform

Modular crypto algorithmic trading platform designed to run natively on Windows Server.

## Features
- Modular monolith architecture
- Paper trading and live trading adaptation (spot trading only, CCXT integration)
- Local SQLite database & Parquet storage
- Machine Learning module incorporating LightGBM (mandatory M9)
- Telegram notifications and alert channels
- Web-based HTML/JS control dashboard
